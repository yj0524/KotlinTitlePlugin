# KotlinTitlePlugin (Paper 1.19.4)

### 사용법

`/titles <player> <title> [subtitle]`

- 쌍따옴표 또는 따옴표로 띄어쓰기 출력 가능 (예 : "a s d f" 또는 'a s d f')

- 색상 코드 적용 가능 (예 : "&aasdf" -> $\textcolor{#55FF55}{\textsf{asdf}}$)
